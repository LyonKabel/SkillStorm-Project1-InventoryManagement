# SkillStorm-Project1-InventoryManagement
First project for skillstorm. An inventory management system.

A Next.js / Java inventory management system featuring data visualization



## Requirements

- Java (v17+ recommended)
- Spring Boot


## Installation
1. Clone the repository:
   ```bash
     git clone https://github.com/LyonKabel/SkillStorm-Project1-InventoryManagement.git

## Starting the frontend
   ```bash
     cd frontend
     npm install
     npm run dev
  ```
## Starting the backend
   ```bash
     cd backend
     ./mvnw spring-boot:run
